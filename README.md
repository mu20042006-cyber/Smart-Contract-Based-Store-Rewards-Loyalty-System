# 📌 Overview

ChainRewards is a decentralized blockchain-based loyalty and rewards platform built using Solidity, Python, and Web3.py.

The system simulates a real-world store rewards program where customers can earn and redeem reward coins while administrators securely manage reward items through smart contracts.

The project combines:
- Smart contracts
- Blockchain analytics
- CLI terminal interaction
- Event listeners
- Security mechanisms
- Blockchain history scanning

---

# 🧠 Smart Contracts

## 📦 StoreRewards.sol

Handles:
- Reward item management
- User registration
- Reward redemption
- Ownership permissions
- Emergency stop system

### Important Functions

```solidity
registerUser(string memory name)
addRewardItem()
batchUpdateRewards()
pause()
resume()
transferOwnership(address newAdmin)
getAdmin()
```

---

## 🪙 RewardCoin.sol

Custom ERC20-style reward token used as loyalty points.

### Features

- Admin-only minting
- Coin transfers
- Balance tracking
- Reward redemption support

# ⚙️ Python Scripts

## 📊 Admin Dashboard

Displays:
- Total reward items
- Total minted coins
- Total transactions
- Top 3 active users

All analytics are generated by scanning blockchain history.

---

## 🚨 Live Alert Listener

Background listener that prints:

```bash
ALERT: A reward purchase just happened!
```

whenever a reward redemption event occurs.

---

## 📁 Balance Snapshot Exporter

Scans blockchain history and exports:

- ETH balances
- Reward Coin balances

into:

```bash
balances_snapshot.csv
```

---

## 🔄 Ownership Transfer Tester

Automated script that:

1. Performs admin action using old admin
2. Transfers ownership
3. Confirms old admin loses permissions
4. Confirms new admin gains permissions
