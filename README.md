ChainRewards — Blockchain Store Rewards System
Overview
ChainRewards is a decentralized loyalty and rewards platform built using Solidity and Python.
The system simulates a real-world store rewards program where customers earn and redeem reward coins while administrators manage reward items securely through smart contracts.
The project includes:


Smart contracts for reward management


Custom reward coin system


Admin dashboard & analytics


Blockchain history scanners


CLI terminal application


User registration/profile system


Live blockchain event monitoring


Security testing


Emergency pause mechanism

How to Run
1. Install Requirements
pip install web3 pandas python-dotenv

2. Deploy Contracts
python scripts/deploy.py

3. Run Terminal Application
python terminal_app/app.py

4. Run Admin Dashboard
python scripts/admin_dashboard.py

5. Run Live Alert Listener
python scripts/live_alert_listener.py
